# Task Management System (Spring Boot + JWT)

## Features
- User Authentication (JWT)
- Role-based access (Admin/User)
- Task CRUD APIs
- Secure REST APIs

## Tech Stack
- Java
- Spring Boot
- Spring Security
- MySQL
- JWT

## Run Project
1. Clone repo
2. Configure DB in application.properties
3. Run Spring Boot
